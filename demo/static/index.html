⚠️ 如果一键安装失败，先执行恢复操作：

  1. 按住 ESP32 上的 BOOT 键不放
  2. 按一下 EN 键然后松开
  3. 松开 BOOT 键

手动步骤：

  1. 下载 MicroPython 固件 →
  2. 安装 Thonny（thonny.org，一路下一步）
  3. 在 Thonny 中：工具 → 选项 → 解释器 → 选择"MicroPython (ESP32)" → 点击"安装或更新 MicroPython"
  4. 将下载的 main.py 拖入 Thonny 文件区，按 F5 运行
